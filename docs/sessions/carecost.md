# CareCost Pitch — Session Learnings

Gold-standard baseline (79% first-pass score). 6-slide pitch deck for healthcare transparency platform.

## Key Pipeline Rules

- **Frontend must be fully data-driven** — no hardcoded CPTs, payers, procedures
- **Sidecar JSON must match HTML** — after convergence, both stay synchronized
- **Registry staleness check** as pre-flight gate before export

## Baseline Artifacts

The `baselines/carecost-79pct/` directory contains the gold standard:
- 13 HTML slide iterations (v1, v2, v3 per slide showing convergence)
- 2 PPTX exports (competitive + main batch)
- README with source map

This baseline demonstrates what 79% quality looks like — use it to calibrate scoring.
